02 Connect Google Sheets OAuth via your own client ID
How to set this up (one-time, ~5 min)
  1. Go to console.cloud.google.com → create a new project (or pick one).
  2. APIs & Services → Library → search "Google Sheets API" → Enable.
  3. APIs & Services → OAuth consent screen → pick External, app name "Meta Matcher", add your Gmail as a test user.
  4. APIs & Services → Credentials → Create Credentials → OAuth client ID → Application type: Web application.
  5. Under Authorised JavaScript origins, add the GitHub Pages URL for this tool.
  6. Copy the Client ID, paste above. Open your Sheet — the URL has /d/SPREADSHEET_ID/edit — copy that ID into the second field.
